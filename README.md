# Análisis de Clientes - ComercioYA

## Descripción del proyecto

Este proyecto tiene como objetivo analizar el comportamiento de clientes de una empresa ficticia de e-commerce llamada ComercioYA, utilizando técnicas de análisis exploratorio de datos (EDA) y regresión lineal.

El análisis busca identificar patrones en variables como edad, ingresos y puntaje de gasto, con el fin de obtener insights que apoyen la toma de decisiones.

---

## Dataset

Se utilizó el dataset **Mall Customers**, que contiene información de 200 clientes, incluyendo:

* Edad
* Ingresos anuales
* Puntaje de gasto
* Género

---

## Tecnologías utilizadas

* Python
* Pandas
* Matplotlib
* Seaborn
* Statsmodels

---

## Análisis realizado

Durante el desarrollo del proyecto se llevaron a cabo las siguientes etapas:

* Carga y limpieza de datos
* Análisis exploratorio (EDA)
* Estadística descriptiva
* Visualización de datos
* Análisis de correlación
* Modelo de regresión lineal

---

## Resultados principales

El modelo de regresión lineal presentó un R² de 0.109, lo que indica que la edad y los ingresos explican solo una pequeña parte del comportamiento de gasto de los clientes.

Se observó que existen distintos perfiles de clientes con patrones de consumo variados.

---

## Conclusiones

El análisis sugiere que el comportamiento de gasto de los clientes no depende únicamente de la edad ni del ingreso. Es probable que existan otros factores relevantes que influyen en las decisiones de compra.

Se recomienda a la empresa explorar nuevas variables que permitan comprender mejor el comportamiento del consumidor.
